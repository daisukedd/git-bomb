# git-bomb
Meu lab pessoal para *estourar* o Git, simular erros reais e dominar workflows avançados antes de subir código para produção.

## Laboratório de Testes
- Cherry-pick: Pescar commits cirurgicamente.
- Stash: Gaveta temporária para pausar tarefas.
- Rebase: Manter o histórico linear e limpo.
- Reflog: Seguro de vida para recuperar o *indeletável*.
- Conflicts: Prática real com marcações `<<<< HEAD`

```
git cherry-pick <hash>    # Pesca a cereja
git stash                 # Salva a bagunça
git pull --rebase         # Sincroniza sem nós
git log --oneline         # Histórico limpo
```
